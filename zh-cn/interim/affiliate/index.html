探索 Deriv.com

Binary.com 已更名为 Deriv.com ，以下是其对合作伙伴的意义

如您所知，我们已经将 Binary.com 更名为 Deriv.com 。这是我们最新承诺的一部分，通过简单、灵活和可靠的解决方案，使所有人都可以方便使用及轻松进行在线交易。

作为我们与您的客户的重要合作伙伴网络的一部分，您可能很好奇并且担心这种转变的意义。我们整理了本指南，以帮助您更加了解品牌重塑，并为您与客户的交流提供支持。

品牌重塑的内容是什么？

品牌重塑活动使我们更坚定地为客户提供灵活、便捷和信心，让客户能随心所欲地交易。这是我们公司为客户提供优异服务的第三个十年成长过程的一部分。

为什么Binary.com 要重塑品牌？

我们在此市场已经有20年，觉得已经到了对客户的承诺给予更新的时候了。品牌重塑增强了我们为客户提供简单、灵活和可靠的在线交易解决方案的决心。

Binary.com 是否已被另一家公司收购？

不，Deriv.com 是由为您带来 Binary.com 的同一团队构建、拥有和管理的。20年的经验让我们对为您服务的能力感到从未有过的强大或自信。

Binary.com 会怎样?

我们会将重点完全放在 Deriv.com 上，Binary.com 网站将逐步淘汰。我们鼓励大家立即移至 Deriv.com 及享受新的体验。客户可在 Deriv.com 使用和访问存储在 Binary.com 的客户账户、资金以及所有内容。

推出了哪些变更？

品牌重塑涉及新名称、徽标、公司颜色和网站。此外，Deriv.com 为客户提供了多种工具，可在 Binary.com 的热门平台 SmartTrader，以及 DTrader、DBot、DMT5 等四大高级平台交易。一些新功能将分阶段提供。

它将如何影响我的客户？

我们知道客户可能会因为此转换而措手不及。因此，我们努力确保以无缝衔接的方式转换成 Deriv.com。作为联盟会员，您在支持他们转换的过程中扮演着关键角色。以下是他们可能提出的一些问题，以及您提供信心保证的答复。

为何我必须转移至 Deriv.com?

品牌重塑活动由我们的客户，就是您所启发。为您服务20年以来，我们觉得能更加了解您的需求。品牌重塑是我们实施改进的在线交易解决方案，以满足您所有需求的机会。

Deriv.com 有什么新增内容?

Deriv.com 在简单、灵活和可靠的环境中为您提供多种交易选项。除了 Binary.com 热门的交易平台 SmartTrader 以外，它还提供 DTrader、DBot 和 DMT5 这三大核心产品。不久以后，以下新功能将会推出:

  • 乘数期权可以为交易者提供更高的利润、同时限制风险

  • P2P 收银台，促进无缝金融交易

  • 亮和暗模式功能，带来更愉快的体验

我可否仍在 Binary.com 交易?

您的资金将同时在 Binary.com 和 Deriv.com 账户中显示，因为两者本质上是同一账户。但是，我们建议您立即试用 Deriv.com，因为我们将逐步淘汰 Binary.com。我们向您保证，您最喜欢的功能仍可以在 Deriv.com 使用。另外，一旦您使用 Binary.com 凭据登录，您在 Binary.com 的账户详细信息、资金和自定义设置将可在 Deriv.com 使用。

如何开始使用 Deriv.com?

很简单。只需前往 Deriv.com 并输入现有的 Binary.com 凭据即可。您将立即能够探索 Deriv.com 的新外观、感受和新添功能。有许多教育资源可帮助您在新的交易空间轻松导航。为了增强您对新环境的信心，请使用模拟账户探索 Deriv.com 。

它将如何转换我们的合伙关系？

作为我们重要的合作伙伴，您无疑会担心自身与 Deriv.com 的合作伙伴关系。请不用担心，我们会尽力为您提供毫无影响的过渡，以便您可以继续建立网络并扩大盈利潜能。

我的联盟会员账户会怎样？

您的 Binary.com 联盟会员账户在 Deriv.com 也可使用。当您的客户在 Binary.com 或 Deriv.com 交易时，您将可以获得佣金。

Deriv.com 的佣金与 Binary.com 的佣金如何区分？

为了您的方便起见，我们将报告分开提供。

要获取每个报告，请转到仪表板，然后选择报告 > 按品牌过滤。然后选择您要查看其佣金的平台。

作为介绍经纪人（IB），此转变将给我带来哪些影响？

无论客户在 Binary.com 的 MetaTrader （MT5）平台或是Deriv.com 的 MetaTrader5（DMT5）平台交易，您都将继续赚取每日佣金。

Deriv.com 是否支持 API 开发人员？

是。如果您已经使用 Binary.com API 开发了自己的交易应用程序，您仍然可以继续使用它。您还可以使用自己的应用收取在 Deriv.com 进行的任何交易的佣金。这些佣金将在您的 Deriv.com 账户中显示。

Deriv.com 是否有付款代理计划?

是。 如果您是 Binary.com 的注册付款代理，那么在 Deriv.com 交易的客户将可以使用您的服务。

我要如何开始推广 Deriv.com?

请放心，您我双方的合作伙伴关系在此品牌重塑过程中至关重要。实际上，我们认为这是您与客户群增强关系的大好机会。且您可以指望我们对您的努力做出响应并给予支持。

我可否立刻开始推广 Deriv.com？

的确。虽然 Binary.com 仍然将存在一段时间，但它会逐渐淘汰。鼓励您的客户转移到 Deriv.com，并让他们在我们新的和改进的平台上交易，从而赚取佣金。

我要如何推广 Deriv.com?

如要推广 Deriv.com，您需要有 Deriv.com 联盟链接。

首先登录 Binary.com 联盟会员仪表板，以获取链接。

选择营销 > 获取媒体代码。然后复制所提供的登陆页面 URL。

登陆页面 URL是您专用的联盟链接，您可用它对您网络内的客户推广 Deriv.com。

你们提供哪些营销支持？

我们非常希望您开始将 Deriv.com 品牌整合到您的所有营销工作中。您可以前往联盟会员仪表板使用我们提供的即用型标语和徽标，以便您能更轻松地开始进行工作。无论您首选的营销渠道是什么，您都会发现 Deriv.com 标语和徽标的格式符合您的特定需求。开始在您的所有渠道使用这些工具，并透过新的 Deriv.com 品牌广为宣传。
